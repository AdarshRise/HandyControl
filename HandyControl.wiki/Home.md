## Welcome to HandyControl
HandyControl is a WPF control library that rewrites almost all native styles and includes more than 60 additional controls (increasingly).
### Requirements
* .Net >= 4.0 or .Net Core >= 3.0
* OS >= win7
* IDE >= vs2019
* C# >= 7.3
