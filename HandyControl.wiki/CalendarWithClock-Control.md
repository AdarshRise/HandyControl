# Usage
First add HandyControl refrence
```
 xmlns:hc="https://handyorg.github.io/handycontrol"
```
then in your xaml
```
<hc:CalendarWithClock/>
```
you can change calendar ui culture:
```
ConfigHelper.Instance.SetLang("en-us");
```
