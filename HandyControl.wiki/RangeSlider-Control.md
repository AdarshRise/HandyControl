# Usage
First add HandyControl refrence
```
xmlns:hc="https://handyorg.github.io/handycontrol"
```
then in your xaml
```
 <hc:RangeSlider Width="400" IsSnapToTickEnabled="True" ValueStart="2" ValueEnd="8"/>
```
| **Available Properti**es |
| ------------------------ | 
| IsSnapToTickEnabled |
| ValueStart |
| ValueEnd |
